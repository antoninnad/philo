# philo_main

Carnet avec mes citations et mes repères vue en philo. Facilement changeable vous avez qu'à modifier dans `data/citation/citation.js`:
```js
  const data = [{
    "name": //node l'auteur,
    "contain": //la citation
  }]

```
pareil pour repères
## Installation
```console
git clone https://github.com/antoninnad/philo_main
firefox index.html
```
